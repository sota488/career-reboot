# Career Reboot

休職中にキャリア迷子になっている人向けの「人生の再始動ガイド」です。診断ではなく、回復から再始動までの人生マップを示すことで、今の現在地を認識し、次のステップへの勇気をサポートします。

## コンセプト

「休職中の現在地診断」
「人生の再始動マップ」

休職中の現在地から、職場復帰・転職・学び直しなどの選択肢へ進む過程を、5つのフェーズで可視化します。ユーザーが「今ここ」を知ることで、焦らず自分のペースで進む道を見つけられるガイドを目指しています。

## 人生の再始動マップ

### フェーズ定義

#### 1. 休息期（テーマ：休む勇気）
心身の回復を最優先にする時期。生活リズムの安定や、無理なく続けられる日常を整えることが大切です。焦らず、できることから始める土台作りに専念しています。

#### 2. 整理期（テーマ：自分を知る）
体力が戻ってきて、次のことを考え始めるタイミング。これまでの人生の棚卸しをしながら、『何がしたいのか』『何は避けたいのか』が少しずつ見えてきる段階です。

#### 3. 探索期（テーマ：可能性を広げる）
進むべき方向の仮説を持ち始めている段階。組織に戻る道、副業、業務委託など複数の選択肢を視野に入れ、相性を小さく試しながら確かめようとしています。

#### 4. 準備期（テーマ：方向性を固める）
『次はこの方向だ』という判断がほぼ固まり、実際の行動に向けて準備を始めている段階。条件整理や必要なスキル習得など、実現に向けた具体的なステップを踏み始めています。

#### 5. 実行期（テーマ：小さく動き出す）
新しい働き方をスタートさせている、または具体的に動き始めている段階。これまでの準備を活かしながら、実際の環境で自分がどう動くか試している真っ最中です。

## 診断について

### 質問カテゴリ（5カテゴリ、各4問、全20問）

1. **回復度** - 心身の安定度を確認
2. **自己受容度** - 自分への許容感と前進する力を確認
3. **自己理解度** - 自分の大切なことや限界を理解できているか確認
4. **未来志向度** - 今後への期待や方向性を確認
5. **行動準備度** - 次のステップへの準備状況を確認

### 質問の特徴

- 能力評価ではなく、感情や状態を確認する文体
- 「〇〇ができていますか」ではなく「〇〇を感じていますか」という問い方
- 5段階評価：まったく当てはまらない → やや当てはまる → とても当てはまる

### 結果判定ロジック

回復度の低さをまず判定し、その後の他カテゴリスコアの組み合わせで現在のフェーズを判定します。

- **回復度 ≤ 10** → 休息期
- **回復度 > 10 且つ**
  - 未来志向度 ≥ 14 && 行動準備度 ≥ 13 → 実行期
  - 自己理解度 ≥ 15 → 探索期
  - 合計スコア（自己受容度+自己理解度+未来志向度+行動準備度） ≥ 50 → 準備期
  - それ以外 → 整理期

## デザイン方針

- **診断サービス** ではなく、**人生の再始動を支援するガイド** として設計
- 過度な装飾は不要
- **安心感、落ち着き、希望** を感じるUIを目指す
- 見出しは「あなたは〇〇型です」ではなく「あなたの現在地は〇〇です」で統一

## ローカル実行

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開きます。

- `/` - ホームページ
- `/diagnosis` - 20問の診断フロー
- `/result` - 診断結果と人生マップ表示
